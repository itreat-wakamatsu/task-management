## Vercel 設定詳細

### ルーティング（vercel.json）

```json
{
  "rewrites": [
    { "source": "/api/(.*)", "destination": "/api/$1" },
    { "source": "/(.*)",     "destination": "/index.html" }
  ]
}
```

- `/api/*` → Serverless Functions
- その他全パス → `index.html`（React Routerで処理）

### 環境変数

| 変数 | 本番 | プレビュー | 説明 |
|------|------|----------|------|
| VITE_SUPABASE_URL | ✓ | ✓ | SupabaseプロジェクトURL |
| VITE_SUPABASE_ANON_KEY | ✓ | ✓ | Supabase公開キー |
| VITE_GOOGLE_CLIENT_ID | ✓ | ✓ | Google OAuth Client ID |
| VITE_BACKLOG_CLIENT_ID | ✓ | ✓ | Backlog OAuth Client ID |
| BACKLOG_CLIENT_SECRET | ✓ | ✓ | Backlog Client Secret（サーバーのみ） |
| VITE_APP_ENV | - | `development` | DEVモード有効化 |

## ネットワーク通信一覧

| 送信元 | 送信先 | プロトコル | 内容 |
|--------|--------|----------|------|
| ブラウザ | Vercel CDN | HTTPS | SPA静的ファイル |
| ブラウザ | Supabase Auth | HTTPS | 認証 |
| ブラウザ | Supabase PostgREST | HTTPS | DB CRUD |
| ブラウザ | Google Calendar API | HTTPS | カレンダー予定取得 |
| ブラウザ | Vercel /api/backlog-token | HTTPS | Backlogトークン交換 |
| ブラウザ | Backlog API | HTTPS | 課題取得 |
| Vercel Serverless | Backlog OAuth | HTTPS | トークン交換（Client Secret使用） |

## 障害影響範囲

| サービス | 停止時の影響 |
|---------|------------|
| Vercel | アプリ全体が利用不可 |
| Supabase | 認証・データ保存が利用不可（今日タブは部分的に動作） |
| Google Calendar API | 今日タブの予定表示が不可（他タブは動作） |
| Backlog API | Backlog連携のみ不可（アプリ本体は動作） |
