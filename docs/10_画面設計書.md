### 1-2. ヘッダー要素仕様

| 要素 | 位置 | サイズ/スタイル | 状態 |
|------|------|--------------|------|
| 日付テキスト | 左上 | 0.75rem, グレー | 常時表示 |
| 日付ピッカー | 日付テキスト横 | date input | DEVモードのみ |
| アプリ名 | 左下 | 1.375rem, bold | 常時表示 |
| DEVバッジ | アプリ名横 | 小・オレンジ背景 | DEVモードのみ |

| ユーザー名▲ | 最右 | 0.875rem | 常時表示 |

### 1-3. タブナビゲーション

- タブ間隔: 均等
- アクティブタブ: 下線あり（`--color-text-primary`）
- 非アクティブ: グレー

│ ID │ タスク名          │ クライアント│ 案件     │ Cat1 │ Cat2 │ 状態 │ 繰返 │開始│期日│操作│

│  43│ 週次MTG資料作成   │ ITreat   │ 基幹システム│ MTG  │      │ 未着手│ ♻  │    │4/5 │✏ 🗑│ ← 期日近

**テーブル列仕様:**

| 列 | 幅 | 内容 |
|----|-----|------|
| ID | 60px | タスクID（Backlogタスクは左に緑Bバッジ） |
| タスク名 | 自動 | タスクタイトル（最大幅で省略） |

| 案件 | 150px | 案件名 |
| カテゴリ1 | 100px | 第1階層カテゴリ |
| カテゴリ2 | 100px | 第2階層カテゴリ |
| 状態 | 80px | 未着手/進行中/完了バッジ |
| 繰返 | 40px | ♻ アイコン（is_recurring=true の場合） |
| 開始日 | 80px | start_date |
| 期日 | 80px | due_date（期日超過時は赤） |
| 操作 | 60px | ✏ 編集、🗑 削除 |
